# 01-why: Strategic Vision

This directory contains strategic vision and market context documents that explain **why** this portfolio website exists and the problems it solves.

## Purpose

These documents answer:
- Why build this portfolio?
- What problems does it solve?
- Who is it for?
- What makes it unique?

## Documents

- **[vision.md](./vision.md)** - Complete vision document covering purpose, goals, target audience, and strategic differentiators

## Relationship to Other Sections

- **Informs** `02-what/` - Vision drives product feature decisions
- **Guides** `05-project-mgmt/` - Strategic goals inform project planning

## Not Specifications

These documents do NOT contain:
- Technical requirements
- Acceptance criteria
- Implementation details
- Architectural decisions

Those belong in `03-how/`.
